# D-Day QA 시나리오 (오너 전용)

> 모든 Codex 작업 머지 후, 데모 전에 반드시 이 7단계를 엔드투엔드로 통과시키세요.
> 하나라도 막히면 데모 전까지 해결.

---

## 사전 준비

- 두 개의 브라우저 창 (또는 시크릿 창) 준비 — Realtime 검증용
- 테스트 계정 2개 로그인
- `.env.local` 에 `ANTHROPIC_API_KEY` 설정 (T7 AI 요약 시연 시)

---

## 시나리오 (분석 리포트 §5 기반)

### Step 1. 신규 가입 & 온보딩 [T6]
- [ ] 새 계정으로 회원가입
- [ ] 부서 선택 + 프로필 생성
- [ ] `/feed` 첫 진입 시 **온보딩 카드 노출**
- [ ] 추천 채널 3개 중 1개 이상 "가입" 클릭 → 해당 카드 제거

### Step 2. 첫 화면 3위젯 확인 [T1]
- [ ] 우측 사이드바에 **커뮤니티 현황 / 요즘 인기 주제 / 오늘의 활발한 채널** 3카드 렌더
- [ ] 숫자·순위가 실데이터로 표시됨
- [ ] 모바일 뷰에서는 숨김 (또는 하단 이동)

### Step 3. Poll + 해시태그 게시글 작성 [T4 + T5]
- [ ] "새 글 작성" → `PostCreateModal`
- [ ] "📊 투표 만들기" 토글 on
- [ ] 질문 + 옵션 3개 입력
- [ ] 본문에 `#데모` `#온보딩` 입력
- [ ] 게시 → 피드에 Poll 카드로 렌더되고 해시태그가 링크로 보임

### Step 4. Realtime 댓글 [T3]
- [ ] 다른 창(계정 B)에서 동일 게시글 열기
- [ ] B 에서 댓글 작성
- [ ] A 창에 **1초 이내 댓글 등장** (새로고침 없이)
- [ ] A 의 알림 벨 배지 증가

### Step 5. 애니메이션 리액션 [T7]
- [ ] 이모지 클릭 시 **튀어오르는 애니메이션** 정상
- [ ] 카운터 숫자 롤링 전환 정상
- [ ] 토글 동작(두 번 클릭 시 취소) 정상

### Step 6. 해시태그 탐색 [T5]
- [ ] `#데모` 링크 클릭 → `/tag/데모` 페이지 이동
- [ ] 방금 작성한 글이 리스트에 보임
- [ ] 다른 태그(`#온보딩`)도 정상 동작

### Step 7. 저장 & 개인화 [T2]
- [ ] 피드에서 "저장" 버튼 클릭 → 아이콘 상태 변경
- [ ] 프로필 페이지 이동 → "저장한 글" 탭에 해당 글 보임
- [ ] 저장 해제 → 목록에서 제거

## 보너스 (시간 있으면)

### Step 8. 온라인 인디케이터 [T7]
- [ ] 두 창 동시 로그인 상태 → Sidebar 하단 "지금 활동 중 · 2명"
- [ ] 아바타에 🟢 점 표시

### Step 9. AI 요약 [T7]
- [ ] 500자 이상 긴 게시글 열기
- [ ] "✨ 3줄 요약" 버튼 클릭 → 로딩 후 요약 박스 렌더
- [ ] 두 번째 클릭 시 즉시 반환 (캐시)

## 3분 시연 스크립트 (발표용)

실제 발표 시 이 순서대로 시연하면 기능 밀도가 최대로 드러남:

```
0:00  로그인 & 온보딩 카드 노출         (T6)
0:20  피드 진입 — 사이드바 3카드 소개   (T1)
0:40  "📊 투표" + 해시태그 입력 시연    (T4 + T5)
1:10  관객에게 QR / URL 공유 → 라이브 투표 참여 (T4)
1:40  두 번째 창에서 댓글 작성 → 실시간 등장 (T3)
2:00  이모지 리액션 애니메이션         (T7)
2:20  해시태그 클릭 → 탐색 피드         (T5)
2:40  저장 → 프로필에서 확인            (T2)
3:00  (선택) AI 요약 버튼 시연         (T7)
```

## 실패 시 대응

- **Realtime 안 옴**: Supabase Dashboard → Database → Replication 에서 `comments`/`notifications` publication 확인
- **위젯 숫자 0**: 시드 데이터 재실행 (`npx tsx packages/db/src/seed.ts`)
- **Poll 투표 반영 안 됨**: 브라우저 캐시 무효화, `polls.getResults` 직접 호출로 확인
- **AI 요약 501**: `.env.local` 의 `ANTHROPIC_API_KEY` 설정 확인 후 서버 재시작
- **해시태그 한글 깨짐**: `/tag/[tag]/page.tsx` 에서 `decodeURIComponent` 처리 확인
